用Github+Hexo搭建博客（三）

发表于 2017-12-13 | 分类于 BLOG教程

搭建这个博客，我用到了Github、Hexo、notepad++和MarkdownPad.
利用notepad++查看和修改各式文本文件，利用MarkdownPad进行博客的编写，通过Hexo这个框架解析生成静态页面，然后部署到托管代码的Github上供大家浏览。
第三部分：关于博客主题的配置和MarkdownPad2的一些Tips
